# Personal Landing Page

## 项目目标

制作一个极简、精致、偏未来感的个人主页，用于线下活动（如 AWS Summit）展示。

用户扫描二维码后进入主页，可以自由选择查看：

* Resume
* LinkedIn
* GitHub
* Featured Projects

目标不是传统 Portfolio，而是数字名片（Digital Business Card）。

核心原则：

5 秒认识我
15 秒了解我的方向
30 秒决定是否连接我

---

## 风格要求

参考风格：

* Claude Design
* Linear
* 极简科技感
* 数字极客风
* 微像素元素

避免：

* 赛博朋克
* 花哨动画
* 大量渐变
* 博客风
* 游戏风

视觉关键词：

Minimal
Calm
Technical
Elegant
Quiet Futuristic

---

## 技术要求

技术栈：

* React
* Vite
* TailwindCSS

单页应用（Single Page）

不要使用后端。

---

## 页面结构

### Section 1 — Hero

内容：
姓名

职业定位：

AI / Data Engineer

副标题：

Speech • Cloud • Full Stack

一句介绍：

Building scalable AI data systems and production pipelines.

按钮：

Resume
LinkedIn
GitHub
Projects

要求：
首屏必须完整展示。

---

### Section 2 — Current Focus

不要使用 About Me。

使用终端风格展示：

CURRENT FOCUS

> Speech Data
> Cloud Infrastructure
> AI Evaluation
> Production Systems

---

### Section 3 — Featured Projects

仅展示 3 个项目。

项目卡片风格：

编号
项目名称
一句描述
技术标签

点击进入详情。

---

### Section 4 — Connect

标题：

LET'S CONNECT

展示：

LinkedIn
GitHub
Email

设计重点：
简洁、留白、易点击。

---

## UI 规范

背景：
深色

圆角：
中等

阴影：
极弱

字体：
标题 → Space Grotesk
正文 → Inter
代码 → JetBrains Mono

动画：
仅允许：

1. 光标闪烁
2. Hover 微位移
3. 极淡背景网格

禁止复杂动画。

---

## 用户体验要求

移动端优先。

页面加载快。

无滚动疲劳。

避免信息过载。

最终感觉：

像一个认真做技术的人，而不是展示技能的人。
